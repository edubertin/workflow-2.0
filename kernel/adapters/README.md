# Kernel Adapters

Adaptadores conectam o kernel a modelos, ferramentas, arquivos, APIs, navegadores, bancos, filas e ambientes externos.

Regras:

- adaptadores ficam nas bordas;
- adaptadores nao definem arquitetura do kernel;
- provedores devem ser substituiveis;
- detalhes proprietarios nao devem vazar para contratos centrais.
